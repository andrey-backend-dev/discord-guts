# Discord Guts

Музыкальные команды теперь используют [`yt-dlp`](https://github.com/yt-dlp/yt-dlp) для получения прямых ссылок на аудио YouTube. 
Это позволяет обойти недавние изменения в скриптах YouTube, которые ломали стандартную загрузку в LavaPlayer 2.2.4.

## Требования к yt-dlp

* Утилита `yt-dlp` должна быть установлена на сервере и доступна в `PATH` (или укажите путь в настройке).
* По умолчанию используется таймаут 20 секунд и команда `yt-dlp`. Эти параметры можно переопределить через `application.yml`:

```yaml
music:
  youtube:
    ytdlp:
      enabled: true
      executable: /usr/local/bin/yt-dlp
      timeout: 30s
```

Если `yt-dlp` недоступен или отключён, бот вернётся к стандартной загрузке LavaPlayer.
